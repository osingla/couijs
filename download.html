This software is provided only through SVN. There is not stamped realease version.
COUIJS includes only four files:
  • couijs.js - This is the COUIJS javascript code.
  • couijs.css - This is the COUIJS Cascading Style Sheet.
  • config.html - File to load from the browser (should and needs to be customized)
  • config.xml - Configuration file (several are provided as example)
Actually, the file config.html can be renamed as you want.
It contains a reference to the XML configuration file, which could named also as you want.

To download these three files, use svn:
svn co https://svn.code.sf.net/p/couijs/code

To play with COUIJS, just follow this simple step:
Open the file
code/config1.html
with your browser (firefox, chrome or IE)
